# Enterprise-Identity-Access-Management-Lab
This project simulates a real-world enterprise Identity &amp; Access Management (IAM) environment for a fictional organisation, Northwind Data Systems. 

It is designed as a mid-to-advanced level implementation, covering end-to-end identity architecture across: 

On-premises Active Directory 

Microsoft Entra ID (Azure AD) 

Hybrid identity integration 

Zero Trust security model 

Identity governance and privileged access 

The lab consists of multi-phase enterprise deployment scenarios, replicating real IAM engineering tasks performed in production environments.
